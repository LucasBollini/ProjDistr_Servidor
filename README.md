# ProjDistr_Servidor
Servidor Para o Projeto Final de Sistemas Distribuidos em Java

Este projeto serve como servidor com capacidade de múltiplos clientes simultâneos. Utiliza TCP e protocolos em JSON.

Funcionalidades:</br>
&emsp;&emsp;• Cadastro, login/logout e alteração de dados de usuários<br>
&emsp;&emsp;• CRUD de Serviços<br>
&emsp;&emsp;• Chat global<br>

Projeto relacionado: https://github.com/LucasBollini/ProjDistr_Cliente
